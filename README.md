# 秘密フォルダⅡ

大容量ファイル(1GB超の動画など)に対応した暗号化フォルダPWA。
複数ファイルをAES-GCM(チャンク方式)で暗号化し、1個の `.himitsu` ファイルにまとめます。
FaceID(パスキー)またはPINで開錠できます。データは端末内で完結し、外部に送信されません。

## 使い方

### つくる
1. 「＋ ファイルを追加」で動画・写真などを選ぶ
2. フォルダ名とPIN(4文字以上)を設定
3. 「FaceIDでも開けるようにする」にチェックすると、このiPhoneのFaceIDで開錠可能に
4. 「暗号化して保存」→ 共有シートの「ファイルに保存」でSDカード等へ

### ひらく
1. 「ファイルを選択」で `.himitsu` を選ぶ
2. 「FaceIDで開く」または PIN入力 →「PINで開く」
3. 一覧から「表示」でその場で再生、「保存/共有」で取り出し

## FaceID開錠について

- iOS 18以降のSafari / ホーム画面追加PWAで利用できます(WebAuthn PRF拡張)
- FaceIDの鍵はこのiPhoneのパスキーに紐づくため、**別の端末ではPINで開いてください**
- iPhoneを初期化・機種変更した場合もPINが頼りです。PINは必ず控えてください

## 技術メモ

| 項目 | 内容 |
|------|------|
| 形式 | `.himitsu` v2 (マジック `HMT2`) |
| 暗号 | AES-GCM 256bit、8MBチャンク方式(大容量対応) |
| 鍵 | ランダム256bitマスターキーを PIN(PBKDF2-SHA256 310,000回) と FaceID(WebAuthn PRF→HKDF) の二重ラップで保存 |
| 改ざん対策 | チャンクごとにAAD(ファイル番号・チャンク番号)を付与 |
| 上限目安 | 1ファイル2GB程度(Safariの実用限界) |

旧「秘密フォルダ」(v1形式)のファイルは旧アプリでそのまま開けます。本アプリはv2形式専用です。
